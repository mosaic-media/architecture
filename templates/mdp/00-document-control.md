# Authority Boundary

MDP-NNN proposes architecture. It does not define it.

Nothing in this document establishes an implementation requirement. If this proposal is accepted, its reasoning is recorded in a Mosaic Architecture Decision, and its outcome is carried by the Architecture Canon and by the supporting specifications that own the affected material.

Per [MDG-001 — Documentation Authority Guide](../../documentation/mdg-001-documentation-authority-guide/02-document-types.md), a proposal must not become permanent reference documentation.
