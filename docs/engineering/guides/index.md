# Guides

Engineering guidelines describe implementation-facing standards, architectural patterns, and quality expectations for Mosaic software.

The current Mosaic Engineering Guidelines are:

- MEG-001 - Go Engineering Standards
- MEG-002 - Event-Driven Runtime
- MEG-003 - Domain-Driven Design
- MEG-004 - Hexagonal Architecture
- MEG-005 - Runtime Architecture
- MEG-006 - Module Platform
- MEG-007 - Storage Architecture
- MEG-008 - Observability
- MEG-009 - Security Architecture
- MEG-010 - Performance Engineering
